# Mistplay Backend Challenge

Note: Search engine is based on the rating of the game.

## To run the program

With express and node.js installed. `cd` into the root directory, and run the command

```
node main.js
```

localhost:3000 is chosen for this program.

### Example test case

http://localhost:3000/search?query=tree&index=0

http://localhost:3000/search?query=good&index=15
